# VLSI-Design-Sprint
An intensive 8-hour VLSI design challenge, executing a complete Silicon-to-Verification flow for a -bit Parity Generator and Checker using the Synopsys Custom Compiler suite. 

Bottom-Up Library Development:m Initiated the project by designing and characterizing fundamental cells, including XOR gates and Inverters. This involved schematic capture, manual transistor sizing, and custom layout implementation for each foundational block.

System-Level Integration: Leveraged the custom-built cell library to architect the 4-bit parity logic, ensuring optimized signal paths for error detection.

Physical Verification (Sign-off): Achieved DRC-clean status and successfully passed LVS (Layout vs. Schematic), guaranteeing 100% structural equivalence between the physical layout and the intended logic.

Post-Layout Characterization: Performed RC Extraction and post-layout simulations to validate timing and signal integrity, accounting for parasitic effects inherent in nanometer-scale interconnects.
